- ctrl + / : 주석처리
- ctrl + f : 텍스트찾기
- ctrl + `(백틱): 터미널 열기/닫기
- ctrl + p: 파일찾기
- ctrl + g: 해당 라인으로 이동
- ctrl + 클릭: 해당 함수정의문으로 이동
- shift + alt + 화살표위아래: 한 줄 복사
- alt + 화살표위아래: 한 줄 라인이동
- ctrl + shift + L: 같은단어 전체선택(전체 수정가능) => ctrl + h: 단어찾기/바꾸기
- ctrl + . : 에러 발생 시 없는 모듈 자동 import
- ctrl + k + f : 자동정렬(모두선택후)
